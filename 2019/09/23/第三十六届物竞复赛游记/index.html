第三十六届物竞复赛游记

发表于 2019-09-23 | 分类于 随笔 | 阅读次数：

$0.$ 前言：

对于本蒟蒻来说，我和很多$OIer$一样，不得不将省一作为自己最终的归宿。出于兴趣也好，为了让自己不用早早沦落至文化课也罢，又再次踏上了物竞之旅。

重点搞物竞的时段也就是高一的暑假了，每天把自己泡在网课与白砖里面泡一下午，虽然效率出奇的低，但看到自己白砖书页的侧棱愈来愈黑，欣慰和愉悦感还是油然而生了。

到了复赛考试前两个礼拜时停止了知识点的学习开始做复赛原卷，只学了力，热，几何光学的我本就知道这次比赛就是去当分母的（后来被LYC神犇纠正了物竞省一是直接按排名前100算的，也就是我连分母这点贡献都没有），于是也就心态旷然，没像班里一些人每天都像在趟油锅的复习。

考试最后一天许爷爷才给我们上完热学，【如果之前没有了解过热学的话，这样囫囵吞枣应该还是拿不到啥分的吧。】像这样想着的我也就没去再听学了三遍的知识点，转去让WY帮我看看白砖上的几何光学题。结果经过一个半小时的研究后被WY怒怼说【你这个光学符号规定是什么啊！凹面镜的半径当然要当负的算啊！】，关系再次破裂，一想到明天还要和他住一个房间，我冷汗直冒。

晚上本来想把白砖好好翻一下，但是有Vtuber直播，身为DD当然要单推了。

$DAY_1^{AM}$

昨晚颓废搞的英语作业也没写，白砖也没看，天泽啊天泽，你怎能如此堕落！难道你以省二的目标也忘了吗？

高铁站上找了YGY和DG，元气骑士走起。

DG他爹好像也来了，滚去问许爷爷三人间怎么办，发现只能和WY住两人间了，冷汗直冒。

高铁的一个半小时里和SQR坐一块儿，发现他已经愉快的把所有知识点复习完在看复赛电学题了，我白砖看着头疼，就给他推了一道养生的力学题一起玩，结果他只花了5分钟就切了，写了和我一样的迷惑性极高的错误答案。

12点左右到了酒店，被告知下午2点前都没有双人间了，于是和WY还要领着行李看考场，吃午饭 红军不怕远征难。

第一次来到中科大，从周边的店面就可以看出是怎样的一所大学：量子咖啡店，量子文具店…..理科气息浓厚，我寻思当你不观察咖啡的时候莫非他还能呈现处在咖啡豆态和咖啡态中的叠加态？

周边还有一家中科大出版社，醒目的摆着一本程力引发众物竞党奔走相告，WY在旁淡然一笑【比起那些大学教材，程力提及的知识点可谓是皮毛中的皮毛，真不知道这么烂的书为啥有人看（有修改）】，结果突然发现与一位从东门出来的格子衫老人擦肩而过，WY转头确认无果后，问我【？？那个人那么老还来中科大，会不会是程老先生？】

看了考场就去吃午饭，没想到在哪里都能看到黄焖鸡米饭。

回来就入住宾馆了，很宽敞，临街但不觉得吵，居然还有咖啡粉供应，不知道比OI住的如家高到哪里去了，我给满分。

$DAY_1^{PM}$

安顿下来后赶快翻出没看完的白砖继续看，WY继续看他的神仙书，本想安逸的度过下午时光。

不到一小时有人敲门，发现是YGY，突然想起来之前中午时候把房间号套了个一元三次方程发给他了，结果被他用CASIO解了（虽然这也是我本意），于是这一下午我白砖就翻了4页纸。。

又不到一小时DG也来了，于是两位父母陪同的可怜人儿就在我们房间开始了愉快的元气骑士联机生活。。

WY看完神仙书后居然掏出了欧姆社的漫话给我们看，真是第一次见到实体装的，惊到了。

就这样混到了5点多，又被喊过去发考场号和准考证号，发现自己考号996， 这暗示了我早上9点搞物竞搞到晚上九点，每周只学6天的快乐生活。

之后又被拉过去看考场。发现自己的302考场正在上【人文主义和思想道德】这种奇怪的科目，而且更奇怪的是左边的301在上【博弈论】，难道这教学楼不应该分文理啥的吗？ 归根结底，这来来回回一小时又浪费在这无意义的长途跋涉中。

回去途中看到了许爷爷发的【为防止大家吃坏肚子，晚上在酒店里包了场，我请客，大家掏钱！】的消息，本来中午的黄焖鸡吃的就很憋屈，看到这条信息还兴奋一下。结果吃完晚饭才发现不仅心里憋屈，胃也憋屈了。。

晚上看了热学和几何光学，和WY关于【考试前一天该不该看这种神仙书】【凹面镜半径是正是负】又吵起来，两位联机打游戏的人形势不对就匆匆退场。看到9点左右发现自己题刷的真心少，居然看完了，于是在WY引导下找了谭雅战记剧场版一起愉(pi)快(pan)地交流，十点半洗了澡就睡了。

WY睡前不忘装弱，我很正经的鼓励了一下。

$DAY_2^{AM}$

早上$6:40$被WY搞醒，搞的困死了，翻身继续睡，但是之后被骚扰的就没睡好了。。

$7:30$起床去楼下吃早饭，这个宾馆是我这辈子见过早餐最丰富的，不仅有关东煮，炒饭春卷一应俱全，吃的巨饱。

之后整理行囊，和WY踏上去考场的路，WY和我探讨了一路日本近现代史，硬是一句关于复赛的话都没讲，弄得我以为正要去二中上学。

坐在考场的最后一排，发现拿卷子是最后一个拿到，收卷子又是第一个被收就很烦。但其实这时候的我反正拿不到省一，被怎样搞都无所谓了。

努力地想去回忆公式，发现早上没睡好，现在只想睡觉。

打铃时候刚好拿到卷子，先把自己能做的题勾出来。1，2力学可以做，第4题怎么第一小题是热学之后是乱七八糟的东西？第五题是光学可以做，然后就没有学过的知识点了，也就是说我把这些题全做对都拿不到省一了。

细细一品，发现这几道题我还是难：WOC第一题静力学我题都读不懂，保持稳定是说椅子不会垮吗？第二题刚体怎么考这个玩意儿，天体今年没了吗？这热学怎么考的那么简单，就算是听了许爷爷速成班的都会做；第五题几何光学怎么那么水（划掉）。

不管，一道题一道题来。再经过迷迷糊糊的40分钟读题后，我明白一个道理：人犯困的时候无论是多么努力的读题，终究是读不懂的。

于是跳了先看第二题，发现怎么这题那么简单？？我会做？？直接上机械能守恒肝啊！之后直接找瞬心套个运动学公式不就没了吗？？

于是我完全忽略了这道题巨额的分值，也忽略了题目要求的轮子钉死的条件，还以为自己要起飞了。

做完了煞笔都会的热学第一小问，发现第二小问怎么也看不懂，于是转去第五题。WOC这公式套的不要太爽！第一题全反射就一高考题，第二题两次近轴成像公式一套就没了，第三题几何关联找找，超越方程丢给CASIO一解就没了，这道全世界都做出来的题也成了我唯一做出来的题。

抬头一看就剩一小时了，我又转过来做第一题，发现题目应该要求灌了水的茶杯最低重心，把定义式写出来求个导就行了，都不用考虑u'v-v'u的正反。但算出来式子极长，第二小问要你求支持力变化量我就知道我肯定没时间解出来了，式子列好就滚粗了。。

就这样交了卷，内心也毫无波动，感觉能拿得分都拿了，怎么说也有80左右吧。考完试的第一时间对了答案，发现第二题标解写了3张纸我就知道我肯定看错题了。。第五题第二小问我踏马1/v=4/3居然写成v=4/3交上去，肯定是没睡醒造成的。。

匆匆问了其他人，出了门第一个见的是SQR，他第一句话是【你做了几道题？我七道都写了字】，第二句话是【我肯定考不到100分以上了】；WY满面荣光就知道他稳了，但还是耐心地听了他FAKE完；有几位巨佬看上去挺难受的，也不知道具体结果。总之，大概就是这样的情况了。

之后也没啥好说的了，高铁站就开始复盘，搞的午饭也没吃（幸好早饭管饱），下午回家就颓颓颓，准备爆肝没写的文化课作业了。

许爷爷高铁站说了【这届好多知识点都没给你们上，他们高三的这次应该考得不错】，但经过这次竞赛的洗礼，很多人没达到内心的目标，就在想着要退役了。我暗地里也寻思着将来的事：明明相比其他巨佬，知识点都没理全的我高三真的还有机会能冲省一吗？

突然又回忆起昨天晚上和WY吵架时候他对我说的话：【别去指挥我考前看什么书！我和你们这些功利心作祟的人不一样！】虽然听上去伤人，狂妄又自大，但你不能否认其中确实有那么几分道理。不论是什么竞赛，什么学科，只有抱着最朴素，最真情的态度去面对，才不会去在意名利上的种种，才会在你所爱的道路上前行。

现在2：12，我坐在这里打上这些文字，希望不论谁看到，都能在自己所爱的道路上不断前行。
